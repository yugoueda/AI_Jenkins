# 全体フロー

詳細はユースケース別の詳細フローを参照する。

```mermaid
sequenceDiagram
    participant User
    participant GitLab
    participant Webhook as Webhook受信サーバー
    participant Jenkins
    participant Agent as エージェント連携

    User->>GitLab: Merge Request作成
    GitLab->>Webhook: Merge Request Hook(opened)
    Webhook->>Jenkins: ビルド/静的解析実行
    Jenkins-->>Webhook: 実行結果通知
    Webhook->>Agent: 初回レビュー依頼
    Agent-->>GitLab: AIレビュー結果コメント

    User->>GitLab: 指摘対応・必要に応じて修正案確認/承認
    GitLab->>Webhook: Note Hook

    User->>GitLab: 全指摘をResolve / push更新
    GitLab->>Webhook: Merge Request Hook(blocking_discussions_resolved=true)
    Webhook->>Jenkins: ビルド/静的解析再実行
    Webhook->>Agent: 再レビュー・ユニットテスト依頼
    Agent-->>GitLab: 追加指摘またはテスト結果コメント
```
